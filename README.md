# Taco Shop Website (HTML Only)

A simple, pure HTML website for a Taco Shop.

## Pages
- Home
- Contact
- Store Timings

## Purpose
This project was created for practice, based on a project idea from the FreeCodeCamp YouTube channel.

## Features
- Semantic HTML
- Basic navigation between pages
- Simple, clean layout (no CSS frameworks or JavaScript)
